
# Practica3 

### Project created with APOS (AVR Project Open Source)

To edit the project settings, open the Makefile and edit the first lines of the file.

En este programa se codifica un contador simple, el cual muestra el resultado en una salidade tipo display de siete segmentos. Los códigos para representar los valores en siete segmentos están almacenados en la SRAM del ATtiny2313.
